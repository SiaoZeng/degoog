# Environment Variables

<details>
<summary><strong>General</strong></summary>

| Variable                    | Description                                   | Default |
| --------------------------- | --------------------------------------------- | ------- |
| `DEGOOG_PORT`               | Port the server listens on                    | `4444`  |
| `DEGOOG_SETTINGS_PASSWORDS` | Set a list of passwords for the settings page | `N/A`   |

</details>

<details>
<summary><strong>Plugins</strong></summary>

| Variable              | Description                                              | Default             |
| --------------------- | -------------------------------------------------------- | ------------------- |
| `DEGOOG_ENGINES_DIR`  | Directory to load custom search engine plugins from      | `data/engines`      |
| `DEGOOG_PLUGINS_DIR`  | Directory to load custom bang command plugins from       | `data/plugins`      |
| `DEGOOG_ALIASES_FILE` | Path to a JSON file defining custom bang command aliases | `data/aliases.json` |

</details>
